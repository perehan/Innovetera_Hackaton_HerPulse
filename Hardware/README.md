# Hardware

This folder contains the hardware components and firmware used in the HerPulse system.

## Components

* ESP32 Microcontroller
* ECG Sensor
* AD8232 Heart Rate Sensor
* DS18B20 Temperature Sensor
* MPU6050 Motion Sensor

## Purpose

The ESP32 collects physiological data from the connected sensors and transmits it to the mobile application for analysis and monitoring.
